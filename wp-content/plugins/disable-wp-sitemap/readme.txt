== Description ==
★★★★★<br>
> This plugin Disable WordPress native default Sitemap automatic creation. 

The sitemap is created automatically by WordPress from version 5.5.
This plugin just give you option to disable it.
Very simple, not administrative panel available. No extra configuration required. Just install and active the plugin. That’s it.
You can activate or deactivate it from your plugin's page.

**Features**
* Simple to use: just activate it.
* Lightweight
* Do the job with a click.

**Why to use this plugin?**
The "Disable WP Sitemap" plugin can be useful in certain situations. 
The XML Sitemap helps to index your website in Google or any other search engine.
If you already have customized and configured your present sitemap you don’t need to do this again for WP Default sitemap.
Having multiple sitemaps in a single website may confuse the search engines.
It is easy to use. Just install and activate it.  No extra configuration required.

== Installation ==

1) Install via wordpress.org

2) Activate the plugin through the 'Plugins' menu in WordPress

or

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
